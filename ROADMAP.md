## 1. Что это и где живёт

**PsyGames** — кроссплатформенное приложение когнитивных тренировок. 62 игры (память, внимание, логика, контроль, счёт, скорость) + уровни/боссы/геймификация.

| | |
|---|---|
| **Репо (единственный источник)** | `donosov999-ai/psygames-native` (private) |
| **Рабочие клоны** | `~/dev/psygames` и `~/Downloads/Code claude/psygames` (оба на одном origin, синхронны) |
| **Стек** | Expo Router + React Native 0.85 + react-native-web; Tauri 2 для десктопа |
| **Веб-деплой** | `psy-games.pro/play` (авто из CI, baseUrl `/psygames-web`) + `/downloads` |
| **Промо-сайт** | `psy-games.pro` — отдельный проект `~/dev/psygames-astro` (Astro, Contabo) |
| **Supabase** | `personal-nzt` (`iuvvheeocobhiothfgei`) — сессии, фидбек, словари |
| **Деплой** | push в `main` + тег `vX.Y.Z` → CI собирает Mac/Win/Android + деплой веба |

**Инфра фидбека тестировщиков (закрытый тест):**
- Кнопка в приложении → текст + скриншот → Supabase таблица `app_feedback` + бакет `feedback-shots` (приватный)
- Пинг Денису: крон `*/2` на 89.116 (`/root/psygames_feedback_notify.py`) → RPC с хеш-гейтом → @Odv999_bot
- Скрины: Edge Function `feedback-shot` (service_role, гейт по sha256 секрета) → бот заливает файлом

## 4. Что висит — по приоритету

### 🔴 Блокеры (решение/действие за Денисом)
- [ ] **Apple Developer Program $99/год** — без него нет iOS-сборки. После оплаты → iOS CI job (сейчас в `build.yml` iOS нет вовсе).

### 🟠 Перед публичным релизом (обязательно)
- [ ] **`FEEDBACK_ENABLED = false`** в `appFeedback.ts:27` — иначе кнопка фидбека висит у всех пользователей, польётся мусор. Сейчас `true` для закрытого теста.
- [ ] **Приёмка вёрстки v1.122.0** — прогон Rulon на его устройстве с крупным системным шрифтом. Фикс проверен статически + tsc + jest + живьём на главной, но именно его Android с 160%-шрифтом из сессии не воспроизвести.
- [ ] **Safe-area Android — добить если нужно.** `viewport-fit=cover` закрывает web/PWA/iOS. Если на Android без выреза лого всё ещё под статус-баром — нужен патч в CI (решение за Денисом):
  - **A** — `windowOptOutEdgeToEdgeEnforcement` (~6 строк, работает сейчас, умрёт на Android 16 → платим дважды)
  - **B** — инсет-листенер → CSS `--sat` (~25 строк Kotlin, переживёт Android 16) ← рекомендация
  - **C** — костыль `Math.max(insets.top, 24)` (2 строки, промахнётся на крупных барах)

### 🟡 Одобрено, не начато
- [ ] **V1 — офлайн-голос (Silero)** для 4 аудио-игр. Одобрено.
- [ ] **V3 — эмбеддинги** для семантических дистракторов semantic-sort. Одобрено.
- [ ] **device_id в фидбеке** — анонимный UUID при первом запуске. Сейчас `person` = профиль («Гость» ×N), при 12 тестировщиках нельзя отличить людей и дубли. Даёт счётчик «сколько разных людей споткнулось об одно место» → приоритезация вместо свалки. ~15 мин.
- [ ] **Аудио-запись в фидбеке** — исходная идея Дениса («нажал и записал: нихуя не понимаю что делать»). MediaRecorder → Supabase Storage → whisper на brainkit 37.60. Пока только текст+скрин.
